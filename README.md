# ♟ Chess Game

A browser-based chess game built with HTML, CSS and JavaScript.

## How to Play
- Open `index.html` in any browser
- Click a piece to select it
- Click a highlighted square to move
- White goes first, players take turns

## Features
- All piece movements (Pawn, Rook, Bishop, Queen, King, Knight)
- Valid move highlighting
- Win detection when King is captured

## Known Limitations
- No check / checkmate detection
- No castling
- No en passant
- No AI opponent

## Built With
- HTML, CSS, JavaScript
- Built with the help of AI tools (ChatGPT + Claude)
